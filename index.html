Git 安装与设置

Ubuntu或是Debian安装git

sudo apt-get install git

或是直接通过源码安装。先从Git官网下载源码，然后解压，依次输入：./config，make，sudo make install这几个命令安装就好了。

$ cd ~/. ssh #检查本机已存在的ssh密钥

如果没有，打开终端输入

ssh-keygen -t rsa -C "邮件地址"

然后三次回车，生成你的ssh公钥，

复制.ssh目录下的id_rsa.pub文件的内容，登录github官网，进入个人设置 -> SSH and GPG keys -> New SSH key。将复制的内容粘贴进key那里，title随便填。

$ ssh -T git@github.com # 测试注意邮箱地址不用改

引言

这几天突然想起github搭建博客这件事，本来自己有阿里云的服务器搭建的wordpress博客。但有一天服务器被人攻击了，好多数据没有还原回来。记得网上说的，搭建博客的人慢慢注意内容，把托管这一块交给其他人，再说我的服务器将来毕业了就恢复原价，将来心里还没谱呢,到时候把更新的博客网站迁移又是个事，所以就想到了github这个平台，可以用来搭建免费的博客，服务器归官方管，只管文字内容就行。非常好的一个平台。

昨天建立仓库，在windows环境安装环境，今天更新debian node.js版本，解决了昨天死活安不成功的问题。
今天选择配置了好长时间的主题，Next主题简洁好用，精简于心，挺棒的。

暂时到这，以后慢慢完善。
